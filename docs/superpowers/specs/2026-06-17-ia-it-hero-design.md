# ia-it 진입 첫화면(Hero) 디자인 스펙

- **작성일**: 2026-06-17
- **프로젝트**: ia-it 브랜드 진입 첫화면 (2026 상반기 트렌디 웹 PC 페이지)

## 1. 개요

가상 브랜드 **ia-it**(AI IT 솔루션 기업, 핀테크/금융 고객 대상)의 웹 PC 진입 첫화면을
2026 상반기 트렌드(시네마틱·의미있는 모션·대담한 미니멀·인터랙티브 3D)에 맞춰 제작한다.

이번 범위는 **진입 첫화면(Above the fold)** — 상단 내비게이션 + Hero 한 화면. 스크롤 없이
보이는 영역에 집중하며, 풀 랜딩 페이지는 후속 작업으로 둔다.

세 가지 무드와 세 가지 인터랙션을 1:1로 페어링하여 **완전히 차별화된 3개 버전**을 제작한다.

## 2. 공통 요구사항

- 데스크톱 우선(PC 페이지) 반응형
- 각 파일은 단독 실행 가능한 자기완결형 HTML (CSS 인라인, JS 인라인 또는 CDN)
- 60fps 목표, prefers-reduced-motion 대응(모션 축소)
- 시맨틱 마크업 + 기본 접근성(키보드 포커스, 대비)

### 공통 Hero 콘텐츠

- **내비게이션**: 로고 `ia-it` · 메뉴(Solutions / AI / Security / Company) · CTA(Contact)
- **헤드라인**: "Intelligence that moves money."
- **서브카피**: AI 기반 금융 인프라 (한 줄 가치 제안)
- **CTA**: 1차 `Get started`, 2차 `View solutions`
- **스크롤 인디케이터**: 하단 중앙

## 3. 세 가지 버전

| 파일명 | 무드 | 인터랙션 | 라이브러리 |
|---|---|---|---|
| `index-aurora.html` | 다크 + 네온 (딥네이비/시안·일렉트릭블루 발광) | 파티클 네트워크 (마우스 반응 노드망) | 순수 Canvas (무의존) |
| `index-flux.html` | 다크 + 미니멀 모노 (차콜/실버 + 포인트 1색) | 플로잉 그라데이션 메시 (셰이더 왜곡) | WebGL 셰이더 (경량, 단일파일) |
| `index-prism.html` | 라이트 + 클린 (오프화이트/블루) | 3D 지오메트리 (회전 결정체, 마우스 반응) | Three.js (CDN) |

### 3.1 index-aurora — 다크 + 네온 + 파티클 네트워크

- 배경: 딥 네이비(#070b1a 계열) 그라데이션, 시안/일렉트릭 블루 발광 액센트
- 인터랙션: Canvas 2D 파티클들이 일정 거리 내에서 선으로 연결, 마우스 근처 노드가 끌려옴/연결 강화
- 무드: AI·데이터·연결을 직관적으로 상징, 가장 화려한 테크 정석

### 3.2 index-flux — 다크 + 미니멀 모노 + 그라데이션 메시

- 배경: 차콜(#0e0e10 계열), 화이트/실버 텍스트 + 단일 포인트 컬러
- 인터랙션: WebGL 프래그먼트 셰이더로 유체처럼 흐르는 그라데이션, 마우스 위치로 왜곡
- 무드: 절제된 고급스러움, 금융 신뢰감, 가장 시네마틱

### 3.3 index-prism — 라이트 + 클린 + 3D 지오메트리

- 배경: 오프화이트(#f6f7fb 계열), 블루 계열 포인트
- 인터랙션: Three.js 와이어프레임 결정체/구체가 회전하며 마우스에 반응
- 무드: 밝고 깨끗, 보수적 금융권 친화, 접근성 우수
